FORMATAÇÃO DO HTML

FORMATAÇÃO DO HTML:

A formatação do HTML é a forma de organizar e estruturar o conteúdo de uma página na internet usando tags que são comandos entre os sinais. Essas tags mostram para o navegador o que cada parte do texto representa como títulos parágrafos imagens e links.Por exemplo existem tags como h1 para títulos p para parágrafos img para imagens e a para links. O html não serve tanto para deixar o site bonito mas sim para organizar as informações de forma correta.Essa organização é importante porque ajuda o navegador a exibir a página corretamente e também facilita que outras tecnologias entendam o conteúdo como mecanismos de busca. A parte visual como cores e fontes geralmente é feita com outra linguagem chamada css enquanto o html cuida da estrutura do site.

ELEMENTOS BÁSICOS DA FORMATAÇÃO

  • h1- até h6- Títulos (do maior ao menor)
  • p- Parágrafo
  • br- Quebra de linha
  • strong- Texto importante negrito
  • em- Enfase itálico
  • ul- / ol- Listas não ordenada / ordenada
  • li- Item da lista
